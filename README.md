## Introduction:
A modular, RESTful backend API built with Node.js, Express and Mongoose, following clean profesiional project structure.. This API manages, products, users and oreders and is designed to be scable, testable and easy to extend.

## Features:
1. Full CRUD operations for:
* Products
* Users
* Orders
2. Database connection using mongoose.
3. Modular routing for clean separation of concerns.
4. Environment variable configuration with dotenv.
5. Custom middleware for request handling.
Git version control with clean commit history.

## Tech Stack:
Runtime: Node.js
Framework: Express.js
Database: Mongoose
Environment: dotenv
Version control: Git and Github

## Testing the API: 
Thunder client.

## Environment Variables:
this project use a .env file to store sensitive information (connction string).

## Sample Data: 
This project includes a data folder containing JSON files with sample documents for products, Users and Orders.
